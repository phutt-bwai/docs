# Zonein AI Assistant Browser Extension

The Zonein browser extension adds context‑aware crypto intelligence to any website. It recognizes the token, contract, pool, or protocol on the page, fetches the right live data, and returns a concise answer you can verify—without leaving your tab. You get the same validated metrics and AI reasoning used in the Zonein web app, delivered as an unobtrusive overlay with session memory for follow‑ups.

# From Page to Answer

## **Context capture**

A Manifest V3 content script inspects safe page signals (URL, DOM hints, selected text) to detect symbols, contract addresses, chart focus, or pool pairs. Only the minimum context needed to answer your question is used; full browsing history is not collected.

## **Intent & scope**

When you ask (right‑click or hotkey), the extension builds a small request: page context \+ your selection \+ the question. A background service worker adds session state (previous Q&A in this tab), signs the request (JWT), and sends it over HTTPS.

## **Data fan‑out**

Zonein’s backend fans out to real‑time APIs: prices/liquidity, on‑chain metrics (holders, volume, transactions), social sentiment, unlock schedules, and deep research history. Fresh values come from hot caches; history from a time‑series store. Conflicting sources are reconciled or labeled as uncertain.

## **Reasoning & ranking**

An AI layer assembles only what matters for the current page, compares to recent baselines, and ranks takeaways. The result includes a TL;DR first, with numbers, links, and a confidence label on demand. Examples: “Token XYZ unlocks 10% on June 30 (medium impact; prior unlocks moved −4–−8% intraday)” or “Whale outflows from top‑10 wallets over 2h; CEX deposits rising.”

## **Overlay delivery**

Answers render in a lightweight in‑page overlay. Session memory keeps context so you can ask, “And tokenomics?” or “Compare to last month,” without re‑selecting. Latency targets: sub‑second for simple checks; heavier analysis streams progressively with the summary appearing first.

## What You Use

**Highlight‑to‑ask.** Select any text (token name, address, pool pair, metric) and right‑click “Ask Zonein.” The selection and page context become the query.

**One‑click prompts.** Common actions appear based on page type: _Unlock schedule_, _Social sentiment_, _Whale flows_, _Pool depth & IL_, _Contract summary_, _Risk check_.

**Actionable cards.** Responses are compact cards: a headline, key numbers, short rationale, and links to sources or the web app for deep dive. Cards can pin, copy, or open in dashboard.

**Session memory.** Follow‑ups in the same tab inherit context. You can reset the session at any time.

**Notifications.** Enable page‑aware alerts: when you revisit a token or protocol, the extension can show a subtle badge if a saved rule triggered (e.g., unlock today, whale outflows \> threshold).

**Permissions & privacy controls.** Granular toggles for site access, data sharing, and telemetry. You can deny access on specific domains, clear local storage, and review what the extension knows about the current session.

# Trust, Performance & Setup

## **Security & privacy**

- Manifest V3 with a background service worker; strict Content Security Policy.
- Minimal context capture; no full browsing history; no private keys ever.
- Encrypted transport (HTTPS/HSTS). API keys stored server‑side; short‑lived JWT per request.
- Local secure storage for session data; editable and clearable by the user.
- Least‑privilege site permissions; per‑site opt‑out.

**Performance**

- Lazy loading of models and data; overlay loads only when invoked.
- Edge/redis caching for hot metrics; offline fallback for recently viewed pages.
- Resource budget designed for low CPU/memory impact; sub‑second p50 for common queries.

**Install & first run**

1. Install from Chrome Web Store and grant site access (recommended: “on click” or “on specific sites”).
2. (Optional) Sign in to sync preferences and alerts with your Zonein account.
3. Choose hotkey and default prompts.
4. Visit a crypto site, highlight text, and right‑click “Ask Zonein.”
